A simple way to represent pairwise relationship between objects. Used to represent large Network of nodes(e.g. Networking, G-maps).

# Type
 * Directed graph
 * Undirected graph


# Representation
  * Adjacency Matrix
  * Adjacency list
  * Adjancency Set


# Algorithms related to graphs
 - DFS
 - BFS
 - Greedy best first search
 - A* search

 - Dijkstra's
 - Bellmon-ford
 - Kruskal's
 - Prims's
